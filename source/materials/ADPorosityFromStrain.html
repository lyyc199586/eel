ADPorosityFromStrain

Porosity calculation from the inelastic strain.

Description

ADPorosityFromStrain computes the porosity, , from the combined inelastic strain, :

Here, is typically computed by ADComputeMultipleInelasticStress using ADViscoplasticityStressUpdate methods.

Example Input Files

[Materials]
  [porosity]
    type = ADPorosityFromStrain
    initial_porosity = 0
    inelastic_strain = strain
    outputs = all
  []
  [strain]
    type = ADGenericFunctionRankTwoTensor
    tensor_name = strain
    tensor_functions = 'volumetric'
    outputs = all
  []
[]
