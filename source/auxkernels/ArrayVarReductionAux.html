ArrayVarReductionAux

Takes an array variable and performs a reduction operation on it (max, min, sum, average) and stores as a standard variable.

Description

This auxiliary kernel performs a reduction operation on the values of an array variable (indexed as ). Reduction operations include max, min, sum, and average. These perform the following operations:
